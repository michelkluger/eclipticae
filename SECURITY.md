# Security Policy

## Supported Versions

Only the latest `main` branch and most recent tagged release are supported with security fixes.

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Report privately by emailing:
- klueger@ecoinvent.org

Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

You can expect an acknowledgement within 5 business days.
